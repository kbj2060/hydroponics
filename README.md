# PlantPoint Web

스마트팜 원격 시스템으로서 농부의 윤택한 생활 을 위해 만들었습니다.
원하는 기기를 서버와 통신하게 하여 원격으로 수동 조종이 가능하고 자동화를 통해 자동 조종이 가능합니다.



## About

### Tech

--------------------------------

- FrontEnd : React, Redux
- BackEnd : Nodejs
- DB : Mysql
- Network : MQTT
- Automation : Python
